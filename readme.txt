Widget for displaying popular posts with thumbnail and post view counter.

== Description ==

Simple and clean styling widget for displaying popular posts with thumbnail and post view counter.

After installed the plugin, WP Popular Posts Widget will be in `Available Widgets` section named `Popular Posts`.

Optional settings for `Title` and `Number of Posts`.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-popular-posts-widget` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
